# pcp-homol-web

Interface web do módulo **PCP** (homologação FANANDRI) — React + Vite + TypeScript.

## Repositórios relacionados

| Repo | Função |
|------|--------|
| [pcp-homol-api](../pcp-homol-api) | API REST (obrigatória em dev) |
| [pcp-homol-docs](../pcp-homol-docs) | Documentação |

## Pré-requisitos

- Node.js 20+
- API rodando em `http://localhost:3000`

## Primeira execução

```bash
cp .env.example .env
npm install
npm run dev
```

- Frontend: http://localhost:5173

## Scripts

| Comando | Descrição |
|---------|-----------|
| `npm run dev` | Servidor de desenvolvimento |
| `npm run build` | Build de produção |
| `npm run preview` | Preview do build |
